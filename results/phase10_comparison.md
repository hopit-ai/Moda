| Source | Size | Description |

| Synthetic Round 1 | 2,910 texts | Generic taxonomy expansions (no benchmark data used) |

**Important**: No benchmark queries, images, or metadata were used in training data generation.
Enriched descriptions were generated by asking an LLM to rephrase product titles of images
already in our training set — this adds query diversity without any leakage risk.
